====== Command Palette ======

This plugin adds a search dialog to allow quickly executing menu entries.

The search dialog can be opened by pressing the keyboard shortcut ''Ctrl+Shift+P'' which can be customized via Zim's key bindings preferences.

{{./command_palette.png}}

**Dependencies:** This plugin has no additional dependencies.
